# Lint Stage Enhancement — Track Context

## Purpose
Enhance local pre-commit hook with Rust code quality checks and file line count enforcement.

## Key Files to Modify
- `package.json` — add lint-staged configuration
- `scripts/check-file-lines.sh` — new file (500-line check)
- `.husky/pre-commit` — auto-generated by husky

## Dependencies
- husky
- lint-staged

## Verification
Run `git commit` with test cases:
1. Clean Rust + short files → should succeed
2. Unformatted Rust → should fail
3. File > 500 lines → should fail
